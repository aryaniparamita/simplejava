Program could be generated from these two steps:
1. Open command line to creditsuisse folder directory
2. Execute the following command: java -jar drawing\out\artifacts\drawing_jar\drawing.jar


The program consists of two files:
1. Main.java --> Input/Output management
2. Canvas.java --> canvas class with functions and utilities

Assumptions:
1. Command is case-sensitive.
2. Diagonal line will be rejected.
3. Points out of canvas boundary will be rejected.
